# 长期记忆

## 用户行为规则（最高优先级）
- **对比/分析 = 纯只读**：用户说"对比"、"分析"、"看看区别"时，**绝对不修改任何文件**，只输出文字报告
- **修改必须显式确认**：只有用户明确说"改"、"更新"、"写入"、"帮我改到文件里"等指令时才能动文件
- **无 git 项目加倍谨慎**：没有版本控制的项目，误操作不可恢复
- **2025-05-25 事件**：因未经授权修改文件被用户严厉警告，此为红线
